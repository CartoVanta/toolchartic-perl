# toolchartic-perl
Perl infrastructure for -toolchartic- toolchains.

## Backward Compatibility Policy

A version number containing an underscore indicates a developer release rather than a stable release.

In such a version, everything before the underscore is the version ID of the most recent stable release.

I make every effort to preserve backward compatibility for features and interfaces that were already present in that most recent stable release.

However, anything added after that stable release and appearing only in developer releases should still be considered in flux.

Such newer additions may be revised, redesigned, or removed before the next stable release.

If what appears before the underscore in a developer release is 0.00, that means there was no stable release before that developer release. In that case, everything should be treated as still in flux.
